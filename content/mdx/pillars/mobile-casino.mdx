## カジノアプリのインストール方法

デバイス別に、カジノアプリのインストール手順を解説します。

### iPhone（iOS）へのインストール

1. **App Store を開く**
2. **カジノ名で検索**（例：「Vera&John」）
3. **「入手」をタップ**
4. **Face ID / Touch ID で認証**
5. **インストール完了後、アプリを開く**
6. **新規登録 or ログイン**

<InfoBox type="tip" title="App Store で見つからない場合">
一部のカジノアプリは App Store で公開されていません。その場合は、Safari からカジノサイトにアクセスし、ブラウザ版をご利用ください。
</InfoBox>

### Android へのインストール

**Google Play Store からの場合：**
1. Play Store を開く
2. カジノ名で検索
3. 「インストール」をタップ
4. インストール完了後、アプリを開く

**APK ファイルからの場合：**

| 手順 | 操作内容 |
|-----|---------|
| 1 | 設定 → セキュリティ → 「提供元不明のアプリ」を許可 |
| 2 | カジノ公式サイトにアクセス |
| 3 | Android アプリのダウンロードボタンをタップ |
| 4 | ダウンロードした APK ファイルを開く |
| 5 | 「インストール」をタップ |
| 6 | インストール完了後、アプリを開く |

<InfoBox type="warning" title="APK ダウンロードの注意点">
APK ファイルは必ず公式サイトからダウンロードしてください。第三者サイトからのダウンロードは、マルウェア感染のリスクがあります。
</InfoBox>

### ブラウザからのアクセス方法

アプリをインストールしたくない場合は、ブラウザから直接プレイできます。

1. **Chrome / Safari を開く**
2. **カジノのURLを入力**（例：verajohn.com）
3. **モバイル版サイトが自動表示**
4. **新規登録 or ログイン**

### ホーム画面にショートカットを追加

ブラウザ版でもアプリのようにホーム画面からアクセスできます。

**iPhone（Safari）の場合：**
1. カジノサイトを開いた状態で共有ボタンをタップ
2. 「ホーム画面に追加」を選択
3. 名前を確認して「追加」をタップ

**Android（Chrome）の場合：**
1. カジノサイトを開いた状態でメニュー（︙）をタップ
2. 「ホーム画面に追加」を選択
3. 「追加」をタップ
